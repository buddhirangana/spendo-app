# Spendo

**Spendo** is a modern, intuitive Android application designed to help users track their daily expenses and manage their personal finances. Built with Kotlin and Firebase, it offers a seamless experience for staying on top of your budget in real-time.

## üöÄ Features

* **Expense Tracking:** Add, edit, and categorize daily spending with ease.
* **Real-time Sync:** All data is securely stored and synced across devices using Firebase.
* **Dashboard Overview:** Get a quick summary of your total balance, income, and expenses.
* **User Authentication:** Secure login and registration to keep your financial data private.
* **Clean UI:** Simple and user-friendly interface designed with Material Design principles.

## üèÅ Getting Started

### Prerequisites

* Android Studio Ladybug or newer.
* A Firebase account to set up your own backend.

### Setup Instructions

1.  **Clone the Repo:**
    ```bash
    git clone https://github.com/buddhirangana/spendo-app.git
    ```
2.  **Firebase Configuration:**
    * Go to the [Firebase Console](https://console.firebase.google.com/).
    * Create a new project and add an Android app.
    * Download your `google-services.json` and place it in the `app/` folder of the project.
    * Enable **Email/Password Authentication** and **Cloud Firestore**.
3.  **Build and Run:**
    * Open the project in Android Studio.
    * Wait for Gradle to sync.
    * Run the app on a physical device or emulator.
